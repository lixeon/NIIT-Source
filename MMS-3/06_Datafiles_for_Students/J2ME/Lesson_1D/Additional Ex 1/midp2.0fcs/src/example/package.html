This directory contains sample programs demonstrating use of the MIDP. Running 'gnumake examples' from build/<platform> will generate MIDlet suites:

  • Games - WormGame, TilePuzzle
  • Demos - Colors, HTTP Test, Properties
  • ManyBalls - Multi-threaded demo with lots of bouncing balls
  • Ticket Auction - Concert Ticket Auction
  • BoxFillIn - Graphical puzzle game
  • Stock - View stocks prices over the internet
